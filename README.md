# FindAndReplace

## How to run

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 15.0.5.

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Features

- Enter a text to search for or generate a random text
- Replace single or all occurrences
- Real-time preview of changes

## Assumptions

- Input text cannot be empty
- Search term cannot be empty when performing replace

## Future Improvements

- Add support for case-sensitive search
- Support for undo replace
- Add search history
- Generate random text with a given length from user input
